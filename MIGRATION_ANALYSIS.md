### ✅ 3. 用户角色与权限（RBAC完整）

```
👤 个人用户 (Individual)
   ├─ assess: 填写健康评估
   ├─ view_own: 查看自身报告
   ├─ dl_own: 下载自身PDF
   └─ view_history: 个人历史追踪
   💾 数据存储: 本地localStorage

   ├─ 个人用户的所有权限
   ├─ submit_org: 提交评估至单位
   └─ view_appt: 查看预约排程
   🔐 验证方式: 成员PIN

📋 部门主管 (Dept Head)
   ├─ 单位成员的所有权限
   ├─ view_dept_okr: 查看部门去识别化OKR
   └─ view_date_report: 按日期查询部门报告
   🔐 验证方式: 主管PIN

🔐 平台管理者 (Admin)
   ├─ 所有权限 (16项)
   ├─ view_org: 全单位KPI
   ├─ manage_okr: 修改OKR参数
   ├─ edit_params: 经济效益参数
   ├─ manage_appt: 管理全部预约

   └─ 首次设定: 三层PIN初始化
   🔐 验证方式: 管理员PIN + Zero Trust
```
